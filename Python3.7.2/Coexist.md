## Modify YUM configuration file

### First

```
vi /usr/bin/yum
```

- Change the `#! /usr/bin/python` in the first line to `#! /usr/bin/python2`

- Press `esc` and enter `:wq` and finally press `enter`

### Second

```
vi /usr/libexec/urlgrabber-ext-down
```

- Change the `#! /usr/bin/python` in the first line to `#! /usr/bin/python2`

- Press `esc` and enter `:wq` and finally press `enter`
